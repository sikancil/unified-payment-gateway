# @indopay/core

Core interfaces, types, and errors for IndoPay.

## Installation

```bash
pnpm add @indopay/core
```

## Key Exports

*   `IPaymentGateway`: The interface all providers implement.
*   `PaymentTransaction`: The standardized transaction object.
*   `PaymentStatus`: Enum for transaction status.
*   `IndoPayError`: Base error class.
